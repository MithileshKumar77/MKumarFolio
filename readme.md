# Mithilesh Kumar | Senior Graphic Designer Portfolio

A high-performance, responsive portfolio website for **Mithilesh Kumar**, a Senior Graphic Designer with 15+ years of experience. This project, developed under the **Aureon Studio** brand, showcases a multidisciplinary design career spanning educational publishing, brand identity, and digital media.

## 🚀 Live Project
**URL:** [https://kalabharti.in/](https://kalabharti.in/)

## ✨ Features
* **11 Specialized Portfolio Categories:** Dedicated sections for Brochures, Logos, Educational Books, Magazine Covers, Packaging, and more.
* **Modern UI/UX:**
    * **Sticky Navbar:** Intelligent show/hide behavior based on scroll position.
    * **Scroll Reveal:** Smooth entrance animations for sections and elements.
    * **Infinite Marquee:** Dynamic scrolling work showcase on the homepage.
    * **Animated Counters:** Real-time counting for years of experience and project milestones.
* **SEO & Social Optimization:**
    * **JSON-LD Schema:** Integrated Person and Occupation structured data for search engines.
    * **Open Graph & Twitter Cards:** Optimized for high-quality social media sharing.
    * **Mobile First:** Fully responsive design using Tailwind CSS utility classes.
* **Lead Generation:** Functional contact form integrated with **Web3Forms** for client inquiries.

## 🛠️ Tech Stack & Tools
* **Frontend:** HTML5, Tailwind CSS (output.css), Vanilla JavaScript.
* **Typography:** Inter font family via Google Fonts.
* **Design Tools Showcased:** * Adobe Creative Suite (Photoshop, Illustrator, InDesign).
    * CorelDRAW, Figma, WordPress, and Canva.

## 📂 Project Structure
```text
├── index.html                # Main landing page (Hero, About, Services, Contact)
├── portfolio.html            # Portfolio hub with 11-category navigation grid
├── portfolio-[category].html # Individual showcase pages (Brochures, Logos, etc.)
├── sitemap.xml               # Search engine index of all project pages
├── assets/
│   ├── css/
│   │   └── output.css        # Compiled Tailwind CSS styles
│   ├── images/               # Optimized WebP assets and video showreels
│   └── icons/                # Multi-platform favicons and webmanifest
└── sitemap.xml               # SEO sitemap for search engines
```

## 🌐 Hosting & Deployment

* **Platform:** Hosted on Cloudflare Pages.
* **Security:** Configured with HSTS and SSL for secure browsing.
* **Performance:** Utilizes lazy loading for images and preconnected Google Fonts to ensure fast PageSpeed scores.
